# 🤖 AI Placement Predictor

An AI-powered Placement Prediction System built using **Python, Streamlit, and Scikit-learn**. The application predicts a student's placement chances using Machine Learning and provides analytics and career guidance.

---

## 🚀 Features

- 🎯 Placement Prediction using Logistic Regression
- 📊 Interactive Analytics Dashboard
- 🤖 AI Career Mentor
- 📈 Data Visualization
- 💡 Career Recommendations
- 📂 Real-world Placement Dataset

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
